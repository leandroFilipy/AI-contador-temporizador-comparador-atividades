# 🏭 Sobre o Projeto

Este projeto implementa um sistema de **controle de atividades** utilizando conceitos de:

🔢 **Contador**  
⏱️ **Temporizador**  
⚖️ **Comparador lógico**  

Muito utilizado em **automação industrial com CLP (Controlador Lógico Programável)**.

---

# 🎯 Objetivo

O objetivo é simular e controlar processos com base em:

✔ Contagem de eventos  
✔ Controle de tempo  
✔ Tomada de decisão baseada em condições  
✔ Automação de processos  

---

# 🧠 Conceitos Aplicados

O sistema utiliza conceitos fundamentais da automação:

### 🔢 Contador
Responsável por contar eventos, como:

- peças produzidas  
- acionamentos de sensores  
- ciclos de máquina  

---

### ⏱️ Temporizador
Controla o tempo de execução de processos:

- atraso para ligar/desligar  
- tempo de operação  
- intervalos entre ações  

---

### ⚖️ Comparador
Realiza decisões com base em condições:

- maior que (> )  
- menor que (< )  
- igual (=)  

Exemplo:

```
Se contador >= 10 → Ativar saída
```

---

# ⚙️ Funcionamento do Sistema

O fluxo básico do sistema segue:

```
Entrada → Processamento (Lógica) → Saída
```

### Exemplo prático:

1️⃣ Sensor detecta evento  
2️⃣ Contador incrementa  
3️⃣ Temporizador controla tempo  
4️⃣ Comparador verifica condição  
5️⃣ Saída é ativada  

---

# 🏗 Estrutura da Lógica

A lógica pode ser representada em **Ladder**, onde:

- 🔘 Entradas → sensores/botões  
- 🔁 Processamento → contadores e temporizadores  
- ⚡ Saídas → motores, luzes, atuadores
